# URL Shortener

A simple URL shortener built with **HTML**, **CSS**, and **JavaScript**. It allows users to input a long URL and generates a shortened version (demo only, no backend functionality).

## Features

- Input a long URL and generate a shortened version.
- Clean and responsive design.
- Demo functionality (no actual URL shortening).

## How to Use

1. Clone the repository or download the files (`index.html`, `styles.css`, `script.js`).
2. Open `index.html` in your browser.
3. Enter a long URL and click "Shorten" to see a demo shortened URL.

## Example

1. Open `index.html` in your browser.
2. Enter a long URL (e.g., `https://www.example.com/very-long-url`).
3. Click "Shorten" to see a demo shortened URL.
